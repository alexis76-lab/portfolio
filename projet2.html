TP : Installation de GLPI sur Linux

Projet d’école sur l’installation et la configuration de GLPI sur un environnement Linux, avec création de la base de données et premier test d’utilisation.

Contexte

Ce projet a été réalisé dans le cadre d’un TP de formation. L’objectif était d’installer GLPI sur Linux, de préparer l’environnement technique nécessaire et de vérifier le bon fonctionnement de la solution à travers une première utilisation de l’outil.

Outils et environnement

Linux GLPI Projet d’école
Objectif

Installer une solution GLPI fonctionnelle sur Linux et comprendre son fonctionnement de base.

Mon rôle

Préparer l’environnement, créer la base de données, installer GLPI et vérifier l’utilisation de l’outil.

Résultat

Une instance GLPI configurée avec sa base de données et un premier test de ticket réalisé.

Type

Projet d’école réalisé en groupe autour de l’installation d’un outil de support IT.

Temps passé

4 heures de travail sur le projet.

Budget

Estimation fictive de 96,16 € bruts en temps humain (1 groupe x 2 personnes x 4 h x 12,02 €), dans le cadre d’un projet d’école.

Début

10 octobre 2025.

Fin

10 octobre 2025.

Déroulement du TP

  • Étape 1 : préparation de l’environnement Linux.
  • Étape 2 : création de la base de données pour GLPI avec phpMyAdmin.
  • Étape 3 : téléchargement et installation de GLPI.
  • Étape 4 : configuration initiale de GLPI.
  • Étape 5 : exploration de l’interface et création d’un ticket.
  • Validation : vérifier que la solution est bien accessible et fonctionnelle.

Travail technique réalisé

  • Préparation du serveur Linux pour accueillir l’application.
  • Création de la base de données nécessaire au fonctionnement de GLPI.
  • Installation de l’application GLPI et configuration initiale.
  • Vérification de l’accès à l’interface web et des premiers paramètres.
  • Création d’un ticket pour valider la prise en main de la solution.

Ce que ce TP m’a apporté

  • Comprendre l’installation d’une solution web sur Linux.
  • Faire le lien entre application, base de données et interface d’administration.
  • Découvrir le fonctionnement de base d’un outil de support et de ticketing.
