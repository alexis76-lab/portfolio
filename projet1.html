TP : Configuration d’un Active Directory

Projet d’école centré sur la mise en place d’un service Active Directory Domain Services, depuis la configuration réseau jusqu’à la validation finale du contrôleur de domaine.

Contexte

Ce projet a été réalisé dans le cadre d’un TP de formation sur Windows Server. L’objectif était de comprendre le rôle d’Active Directory Domain Services puis de suivre une procédure complète de configuration d’un serveur jusqu’à sa promotion en contrôleur de domaine.

Outils et environnement

Windows Server Active Directory AD DS TP d’école
Objectif

Mettre en place un Active Directory fonctionnel et comprendre le rôle d’AD DS dans une infrastructure.

Mon rôle

Configurer le serveur, installer le rôle AD DS, promouvoir la machine et vérifier le bon fonctionnement final.

Résultat

Un contrôleur de domaine configuré avec les prérequis réseau et les vérifications de base du service.

Type

Projet d’école en groupe autour de la configuration d’un Active Directory.

Temps passé

4 heures de travail sur le projet.

Budget

Estimation fictive de 192,32 € bruts en temps humain (2 groupes x 2 personnes x 4 h x 12,02 €), dans le cadre d’un projet d’école.

Début

10 octobre 2025

Fin

10 octobre 2025

Déroulement du TP

  • Partie 1 : comprendre le rôle d’Active Directory Domain Services.
  • Étape 1 : configurer une adresse IP statique sur le serveur.
  • Étape 2 : installer le rôle AD DS.
  • Étape 3 : promouvoir le serveur en contrôleur de domaine.
  • Étape 4 : vérifier la configuration et le bon fonctionnement global.
  • Validation : s’assurer que le service est correctement déployé et exploitable.

Travail technique réalisé

  • Préparation du serveur et vérification des prérequis réseau.
  • Définition d’une adresse IP statique pour garantir la stabilité de l’infrastructure.
  • Ajout du rôle Active Directory Domain Services depuis Windows Server.
  • Promotion du serveur en contrôleur de domaine via l’assistant de configuration.
  • Vérification finale de l’installation et du bon état du service Active Directory.

Ce que ce TP m’a apporté

  • Mieux comprendre la logique d’un annuaire Active Directory dans un SI.
  • Suivre une procédure d’installation technique et la valider étape par étape.
  • Faire le lien entre configuration réseau, services Windows Server et administration de domaine.
